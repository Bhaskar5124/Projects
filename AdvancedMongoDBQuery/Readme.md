## Application needed to Run Project - 
1. VS Code
2. MongoDB Compass

# Your Local Machine should have installed Node.js Globally.

# How to Run the Proejct - 
1. Open AdvancedQuery folder in VS Code
2. Do run Command: 'npm install'
3. Create .env file configuring 'PORT' and 'MONGO_URI'
4. Run Command: 'npm run seed' to seed the data to the MongoDB database.
5. Run Command: 'npm start' for launching and testing for Testing

It can also use MongoDB Atlas, If want to use MongoDB Atlas, configure 'MONGO_URI' in .env file accordingly. MongoDB Compass Preferred.
